## Overview
Telco USSD Assist is an MCP server that exposes Ghanaian telecom USSD codes (MTN, Telecel, AirtelTigo, Globacom) as callable tools for AI assistants. The project aims to provide a single source for USSD lookups instead of scattered documentation.
