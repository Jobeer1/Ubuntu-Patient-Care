Print to PDF

Press Ctrl+P (Windows) or Cmd+P (Mac) to open the print dialog.

  1. Set Destination to "Save as PDF"
  2. Set Paper size to "A4"
  3. Set Scale to "Fit to page" or "100%"
  4. Enable "Background graphics" for the blue border
  5. Click Save
