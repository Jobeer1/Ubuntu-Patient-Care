## WHAT THIS PROJECT DOES

FarmerConnect MCP provides 6 specialized tools for AI agents assisting farmers:

1. **calculate_agro_metric** - Land area, plant density, yield, conversions
2. **get_weather_now** - Real-time temperature and precipitation
3. **forward_geocode** - Place name to coordinates
4. **reverse_geocode** - Coordinates to place name
5. **get_crop_info** - Encyclopedic crop information
6. **get_current_datetime** - Server time

**Use Case:** AI agents help farmers make informed decisions about:
- When/where to plant
- What to plant (based on location)
- Yield expectations
- Weather conditions
