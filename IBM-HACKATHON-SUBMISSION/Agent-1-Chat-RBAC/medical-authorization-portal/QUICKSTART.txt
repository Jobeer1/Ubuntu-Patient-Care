HOW TO START (3 EASY STEPS):
============================

STEP 1: Navigate to the folder
   cd "medical-authorization-portal"

STEP 2: Install dependencies (first time only)
   - Windows CMD: setup.bat
   - PowerShell: .\start.ps1
   - Or manual: pip install -r requirements.txt

STEP 3: Run the app
   python app.py

That's it! Open your browser to: http://localhost:5000

TROUBLESHOOTING:
================

Q: "Port 5000 already in use"
A: Change port in app.py line 48:
   app.run(debug=True, port=5001, threaded=True)

Q: "Module not found"
A: Install dependencies:
   pip install -r requirements.txt

Q: "Database is locked"
A: Close other instances and restart

Q: "Template not found"
A: Ensure templates/ folder exists with all .html files

Q: "CSS not loading"
A: Ensure static/css/style.css exists
