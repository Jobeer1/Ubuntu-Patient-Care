## 🎬 USER WORKFLOWS

### Workflow 1: Admin Creates New Role
```
[Admin Dashboard]
    ↓
    Click "+ Create Role"
    ↓
    [Modal Opens]
    ├─ Enter: "Pathologist"
    ├─ Select: Permissions
    └─ Click: Save
    ↓
    ✅ Role Created & Visible
```

### Workflow 2: Doctor Accesses Patient Studies
```
[Login Page]
    ↓
    Enter: Credentials
    ↓
    [Dashboard]
    ↓
    Auto-redirect to PACS
    ↓
    [Patients Page]
    ├─ See: Only assigned patients
    └─ Access: Studies for those patients
    ↓
    ✅ View/Download Reports
```

### Workflow 3: Admin Grants Patient Access
```
[Admin Dashboard]
    ↓
    Click: "Roles & Permissions" tab
    ↓
    Click: "🔒 Patient Access"
    ↓
    Click: "Grant Access" button
    ↓
    [Modal Form]
    ├─ Select: Patient ID
    ├─ Select: User
    ├─ Choose: Access Level
    └─ Set: Expiration (optional)
    ↓
    Click: "Grant"
    ↓
    ✅ Access Granted
```

## 🔄 QUICK START

### Access the System
```
URL: http://localhost:8080
Admin Dashboard: http://localhost:8080/admin
```

### Create Your First Role
```
1. Go to Admin Dashboard
2. Click "🎭 Roles & Permissions" tab
3. Click "+ Create Role"
4. Fill in role details
5. Select permissions
6. Click "Save Role"
```

### Assign Role to User
```
1. Go to "👥 Users" tab
2. Click "Edit" on a user
3. Select role from dropdown
4. Click "Save"
```

### Grant Access to Patient
```
1. Go to "🔒 Patient Access" tab
2. Click "Grant Access"
3. Select patient and user
4. Set access level and expiration
5. Click "Grant"
```
