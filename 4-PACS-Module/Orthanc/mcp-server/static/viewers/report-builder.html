Report Builder Help

Getting Started

  • Select a study from the dropdown
  • Choose a report template
  • Use auto-populate to insert data
  • Edit the report text as needed
  • Use speech dictation for hands-free input
  • Preview and export when complete

Keyboard Shortcuts

  • Ctrl+B - Bold text
  • Ctrl+I - Italic text
  • Ctrl+U - Underline text
  • Ctrl+S - Save report
  • Ctrl+P - Print report
  • Ctrl+D - Start/Stop dictation

Speech Dictation

  • Click "Start Dictation" or press Ctrl+D
  • Speak clearly into your microphone
  • Say "period" for punctuation
  • Say "new paragraph" to start a new paragraph
  • Click "Stop Dictation" when finished

Auto-Populate

  • Select a study first
  • Click "Insert" next to any field
  • Data will be inserted at cursor position
  • Edit the inserted text as needed
