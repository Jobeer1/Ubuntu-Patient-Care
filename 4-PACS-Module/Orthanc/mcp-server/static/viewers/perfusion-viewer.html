Perfusion Viewer Guide

Features:

  • Dynamic Series: Navigate through time frames with the slider
  • TIC Analysis: View time-intensity curves and statistics
  • Perfusion Maps: Display CBF, CBV, and MTT parametric maps
  • Blood Flow: Estimate cerebral blood flow per region
  • ROI Tools: Draw circular or rectangular ROIs for analysis
  • Export: Generate clinical reports with findings

Keyboard Shortcuts:

  • ← → Previous/Next frame
  • Space Play/Pause animation
  • R Reset view
  • E Export report
